## 說明

安裝軟體的部分我就沒有詳細介紹，以下是我使用到的工具跟語言。

開發工具：
1. Visual Studio 2017 (可使用其他版，只要有支援 ASP.NET Core 開發就可以。)
2. Nodejs 6.9.x 以上版本 及 npm 3 以上版本

開發語言：
1. .NET Core 使用 C#
2. Angular 4 使用 TypeScript

下載範例後記得在專案目錄安裝 npm package

``` batch
npm install
```
